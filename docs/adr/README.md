# Architecture Decision Records (ADR)

ADR documents capture architecture-level decisions and their rationale.

## Status values

- `Proposed`
- `Accepted`
- `Rejected`
- `Superseded`

## Naming

- Format: `NNNN-short-kebab-title.md`
- Example: `0001-system-architecture-and-boundaries.md`
