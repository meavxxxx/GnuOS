# Glossary

## ABI

Application Binary Interface. Contract for calling conventions, symbol names,
and binary compatibility between components.

## ADR

Architecture Decision Record. Versioned document capturing an architecture
decision, context, and consequences.

## IPC

Inter-process communication. In GNU OS roadmap this includes message passing,
capability transfer, and shared-memory mechanisms.

## PT_INTERP

ELF program header that points to a dynamic loader path (for example
`/lib/ld-gnuos.so.1`).

## TLS

Thread-Local Storage. Per-thread data access mechanism used by libc/runtime.
